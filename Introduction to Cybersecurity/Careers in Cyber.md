# Overview

Cybersecurity is one of the fastest-growing industries, offering a wide variety of career opportunities. From protecting organizations against cyber threats to ethically hacking systems, there is a role suited to every interest and skill set.

This room introduces some of the most common cybersecurity career paths, explains their responsibilities, and provides learning paths to help beginners start their journey.

---

# Why Choose a Career in Cybersecurity?

Cybersecurity professionals are in high demand due to the increasing number of cyber threats worldwide.

Some of the main benefits of pursuing a cybersecurity career include:

- High-paying job opportunities
- Strong career growth
- Exciting and challenging work
- Opportunity to specialize in different domains
- Millions of unfilled cybersecurity positions globally

Whether you're interested in offensive security, defensive security, digital investigations, or malware analysis, cybersecurity offers numerous career options.

# Task 2 - Security Analyst

A **Security Analyst** is responsible for monitoring an organization's systems and identifying potential security threats.

They continuously review alerts, investigate suspicious activities, and work with other teams to strengthen security controls.

### Responsibilities

- Monitor security alerts and events
- Investigate suspicious activity
- Identify security risks
- Prepare security reports
- Recommend security improvements

### Recommended TryHackMe Learning Paths

- Pre Security
- Cyber Security 101
- SOC Level 1

---

# Task 3 - Security Engineer

A **Security Engineer** focuses on designing, implementing, and maintaining security solutions across an organization's infrastructure.

They work closely with security analysts and other IT teams to improve the organization's overall security posture.

### Responsibilities

- Implement security controls
- Monitor networks and systems
- Fix vulnerabilities
- Maintain security infrastructure
- Improve system security

### Recommended TryHackMe Learning Paths

- SOC Level 1
- Jr Penetration Tester
- Offensive Pentesting

---

# Task 4 - Incident Responder

An **Incident Responder** investigates security incidents while they are occurring.

Their objective is to contain attacks quickly, minimize damage, recover affected systems, and prevent similar incidents in the future.

### Responsibilities

- Investigate security incidents
- Contain cyberattacks
- Recover compromised systems
- Develop incident response plans
- Document incidents and lessons learned

### Recommended TryHackMe Learning Path

- SOC Level 1

---

# Task 5 - Digital Forensics Examiner

Digital Forensics Examiners investigate cyber incidents by collecting and analyzing digital evidence.

Their work may involve assisting law enforcement agencies or helping organizations determine how security incidents occurred.

### Responsibilities

- Collect digital evidence
- Analyze compromised systems
- Recover deleted files
- Prepare forensic reports
- Support legal investigations

---

# Task 6 - Malware Analyst

Malware Analysts specialize in understanding malicious software.

They examine malware samples to determine how they function and develop techniques for detecting and preventing future attacks.

### Responsibilities

- Perform static malware analysis
- Perform dynamic malware analysis
- Reverse engineer malicious software
- Document malware behavior
- Prepare technical reports

---

# Task 7 - Penetration Tester

Penetration Testers, also known as Ethical Hackers, simulate cyberattacks to identify vulnerabilities before malicious attackers can exploit them.

They assess the security of networks, systems, and applications, then provide recommendations to improve security.

### Responsibilities

- Perform penetration testing
- Conduct vulnerability assessments
- Test web applications
- Assess network security
- Prepare professional security reports

### Recommended TryHackMe Learning Paths

- Jr Penetration Tester
- Offensive Pentesting

---

# Task 8 - Red Teamer

Red Teamers simulate realistic cyberattacks to evaluate an organization's ability to detect and respond to threats.

Unlike penetration testers, who mainly focus on identifying vulnerabilities, Red Teamers emulate real attackers by attempting to compromise systems while avoiding detection.

### Responsibilities

- Simulate advanced cyberattacks
- Test detection and response capabilities
- Maintain persistence during engagements
- Evade defensive controls
- Provide recommendations for improving security

### Recommended TryHackMe Learning Paths

- Jr Penetration Tester
- Offensive Pentesting
- Red Teamer
